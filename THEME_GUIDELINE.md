# LIORA PLACEMENT PLATFORM — THEME GUIDELINE v2.0

## Brand Positioning

Liora is a behavioral health placement marketplace helping users discover:

* Detox Facilities
* Residential Treatment Centers
* Mental Health Facilities
* PHP Programs
* IOP Programs
* Recovery Communities

The platform should feel like:

* Zillow
* Senior Living Directory
* Healthcare Placement Platform

NOT:

* SaaS Dashboard
* Startup Landing Page
* Crypto Website
* Dark Theme Product

---

# 1. Core Design Principles

### Trust First

Users are making life-changing healthcare decisions.

The UI must feel:

* Safe
* Professional
* Clean
* Reliable

### Information First

Content should always be more important than decoration.

### Directory First

Users browse listings.

Everything should support:

* Searching
* Comparing
* Contacting

### Premium Simplicity

Premium does NOT mean flashy.

Premium means:

* Clean spacing
* Excellent typography
* Subtle shadows
* Consistent design

Every page must share:

* Same typography
* Same buttons
* Same spacing
* Same card system
* Same hover effects
* Same animation timing

No page should feel like it belongs to a different product.

# Final Goal

Create a premium, trustworthy behavioral health marketplace that visually resembles Zillow and professional healthcare placement platforms while maintaining a modern and highly responsive user experience.
